# Design Tokens — Mat Analytics

Variables CSS definidas en `:root` y usadas en todas las páginas.

## Paleta de color

| Token | Valor | Uso |
|-------|-------|-----|
| `--red` | `#8B0101` | Acento principal, CTAs, highlights |
| `--red-dark` | `#6B0101` | Hover states oscuros |
| `--red-hover` | `#A50101` | Hover de botones primarios |
| `--red-subtle` | `rgba(139,1,1,0.08)` | Fondos de tags |
| `--black` | `#1E1E1E` | Negro corporativo, fondos dark |
| `--black-soft` | `#2A2A2A` | Negro suavizado |
| `--gray-text` | `#3A3A3A` | Texto principal en fondos claros |
| `--gray-mid` | `#757575` | Texto secundario, iconos |
| `--gray-trace` | `#999999` | Texto terciario, labels |
| `--gray-light` | `#D9D9D9` | Bordes en light |
| `--gray-bg` | `#F7F7F7` | Fondos de secciones alternas |
| `--white` | `#FFFFFF` | Texto sobre dark, fondos |

## Tipografía

| Token | Valor |
|-------|-------|
| `--font` | `'Inter', -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif` |

Pesos usados: 300, 400, 500, 600, 700, 800

## Espaciado y forma

| Token | Valor | Uso |
|-------|-------|-----|
| `--radius` | `6px` | Botones, inputs, tags |
| `--radius-lg` | `12px` | Cards, modales, dropdowns |
| `--max-w` | `1160px` | Ancho máximo del contenedor |

## Sombras

| Token | Valor |
|-------|-------|
| `--shadow` | `0 4px 24px rgba(0,0,0,0.10)` |
| `--shadow-lg` | `0 12px 48px rgba(0,0,0,0.18)` |

## Animaciones

| Token | Valor |
|-------|-------|
| `--transition` | `0.22s ease` |

## Bordes

| Token | Valor |
|-------|-------|
| `--border` | `rgba(255,255,255,0.08)` — para fondos dark |
| `--border-light` | `#E5E5E5` — para fondos claros |
